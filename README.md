# Rohan Oddam — Portfolio Website

This is my personal portfolio website built using HTML and CSS.

I created this project while learning frontend development during my engineering journey. The goal was to build something clean, modern, and realistic instead of relying on heavy frameworks or complicated designs.

The website focuses on:
- responsive layouts
- modern UI design
- clean structure
- better CSS practice
- building real projects while learning

## Features

- Responsive dark-theme design
- Glass-style UI elements
- Smooth hover effects
- Minimal and clean layout
- Beginner-friendly frontend project structure
